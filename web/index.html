Write a Tufte document once. Ship both formats.

A Typst template in the spirit of Edward Tufte. One .typ source compiles to a print-ready PDF (via the marginalia package) and a web page in canonical tufte-css markup. Sidenotes, margin figures, full-width plates, epigraphs. Five styles, both formats, from the same body.
